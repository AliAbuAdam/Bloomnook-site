# Bloom Nook

Лендинг и магазин розничной продажи луковиц цветов с заказом через маркетплейс Ozon.
Свёрстан на **Next.js (App Router) + TypeScript** по дизайну из Claude Design (Linear WEB-3).

Фирменный стиль — строгий минимализм в тёмно-зелёном (`#183512`) и белом, шрифты
Playfair Display (заголовки) + Manrope (текст), фирменная ботаническая линейная графика.

## Страницы

| Маршрут     | Описание                                                              |
| ----------- | -------------------------------------------------------------------- |
| `/`         | Главная: hero, категории, хиты сезона, акция «Осенняя посадка» с живым таймером, «Как это работает», преимущества, отзывы, FAQ, подписка |
| `/shop`     | Магазин: фильтры, сетка товаров, сортировка, пагинация                |
| `/product`  | Товар: галерея, фасовка, окраска, счётчик, табы, похожие товары       |
| `/cart`     | Корзина: позиции, промокод, сводка заказа                             |

Заголовок, подвал и информационная плашка — общие для всех страниц (в `app/layout.tsx`).

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # продакшен-сборка
npm run start    # запуск собранного билда
```

## Структура

- `app/` — маршруты (App Router) и глобальные стили (`globals.css` с дизайн-токенами, шрифтами, keyframes).
- `components/` — переиспользуемые компоненты: шапка/подвал, карточка товара, ботанические мотивы (`MotifSprite` + `Motif`), таймер акции (`SeasonalPromo`), аккордеон FAQ, иконки.
- `lib/data.ts` — данные каталога (товары, категории, отзывы, FAQ и т.д.) и форматирование цен.
- `public/` — логотипы (`header_logo.svg`, `footer_logo.svg`).

Исходный дизайн-бандл сохранён в `.design-src/` (вне сборки) для справки.

## Интеграция с Ozon

Кнопки «Заказать на Ozon» / «Оформить на Ozon» — основное целевое действие. Сейчас это
заглушки; при подключении нужно подставить реальные ссылки на карточки товаров в Ozon.
